## ZilPay mobile

ZilPay is a mobile web browser that provides access to Web3 that use the Zilliqa blockchain.

### Building Locally

installing dependencies:
```bash
$ yarn install
# or
$ npm install
```

server project:
```bash
$yarn ios
# or
$npm run ios
```

web:
```bash
$yarn web
#or
$npm run web
```

android:
```bash
$yarn android
#or
$npm run android
```
